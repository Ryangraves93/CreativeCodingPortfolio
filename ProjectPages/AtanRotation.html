| Description

A sketch building on previous mapping functionality. For this implementation we developed on the previous sketch by implementing atan2 which is the tangent inverse. The grid is split into multiple objects which all deal with their own rotation.

Code Link

| Functionality

  • Calculates theta,the rotation angle by passing in the mouse position y & x
    • var theta = atan2(mouseY - this.y,mouseX - this.x)
  • Rotates the classes inner circle around the angle
    • rotate(theta)
  • Maps the size and the offset to distance between it's x & x position and the mouse position x & y.
    • var size = map(dist(this.x,this.y,mouseX,mouseY),0,width,5,15)
    • var offset = map(dist(this.x,this.y,mouseX,mouseY),0,width,0,10)
