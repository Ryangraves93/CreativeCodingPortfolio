| Description

Implemented a font library and created an algorithm to distribute out words using the vector positions from the library for wordss

Code Link

| Functionality

  • Uses a type font library that provides a vector library for letter positions.
  • Provides a slider to scale number of segments for finer HSB values.
  • Samples a path which is input and then uses a conditional statement based on the next array content to provide functionality
    • switch (fontpathArr[i].commands[j].type) {
      case "M":
      beginShape();
      vertex(fontpathArr[i].commands[j].x, fontpathArr[i].commands[j].y);
      break;
      case "C":
      curveVertex(fontpathArr[i].commands[j].x, fontpathArr[i].commands[j].y);
      curveVertex(fontpathArr[i].commands[j].x1, fontpathArr[i].commands[j].y1);
      curveVertex(fontpathArr[i].commands[j].x2, fontpathArr[i].commands[j].y2);
      break;
      case "L":
      vertex(fontpathArr[i].commands[j].x, fontpathArr[i].commands[j].y);
      break;
      }
  • The functionality is scalable as the method takes any number of words or letters
