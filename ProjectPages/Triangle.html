legends

Description

A set of subdivided triangles using a 2-dimensional array. Creating a grid of vertices, I then subdivided these points down to 3 triangles and then passed the data into a function to render the triangles.

Functionality

  • Generates an equally distributed amount of points across each row as vectors which then get placed into a 2-D array .

    • RowPoints[j].push( createVector(((i * scaler)* modifier + offset), (j * scaler) * modifier))
  • Taking the modulo of the row number to determine what order to draw the triangles in.

    • DrawTriangles(RowPoints[i + 1][j], RowPoints[i][j], RowPoints[i][j + 1])
      DrawTriangles(RowPoints[i][j + 1], RowPoints[i + 1][j + 1], RowPoints[i + 1][j])

  • Then in the DrawTriangles function, runs a call to get the mid-point of the data passed in, then draws three triangles using the given data.

    • function CalculateMid(pointA, pointB, center)
      let mid = createVector((pointA.x + center.x + pointB.x) / 3, (pointA.y + center.y + pointB.y) / 3)
      return mid;
