Description

A set of translated object that each draw vertex points around an angle centered on each object. The sketch takes previous implementations of inverse tangent rotations and applys it to vertice points. The sketch will have a different pattern each time and based on randomised numbers within the sketch.

Functionality

  • Creates a variable on setup to determine if the pattern is consistent or random on each object.
    Then depending on if it's true or false, sets the variable randrange to be consisten, or random each time.

    • bConsistent = round(random(0, 1))

    • bConsistent ? randRange = patternRandRange : randRange = random(1, 15)
      Squares.push(new RotatingSquare(stepX * i, stepY * j, stepX, stepY, i, randRange))

  • The sketch then runs the render function which passes an incrementing variable in to create the angle at which each vertex is drawn.
